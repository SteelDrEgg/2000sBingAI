### How to use

Download source code and unzip
```shell
wget https://github.com/SteelDrEgg/2000sBingAI/archive/refs/heads/master.zip
unzip master.zip
```

cd into directory then download dependencies
```shell
pip3 install -r requirements.txt
```

Run
```shell
python3 main.py
```
